# @sequential/desktop-api-client

HTTP client for Sequential-OS API operations with automatic URL construction and error handling.

## Installation

```bash
npm install @sequential/desktop-api-client
```

## Usage

```javascript
import {{ main }} from '@sequential/desktop-api-client';

const result = main(/* args */);
```

Main function/class provided by this package.

// Basic usage
